## What is SYF Core?

SYF Core is the minimal, law-bearing kernel of the Systemic Foundation.  
It is **not software**, **not a product**, and **not a framework**.  
It is a **mathematical law made executable**.

SYF Core implements:
- **SyFF** — Systemic Fire Formula
- **R** — Systemic Ratio (invariant output)
- **FirePlank™** — Safety floor (continuity constraint)
- **Core Axioms** — No governance, no oracle, no feedback, no intent

## What SYF Core Is NOT

- Dizer  
- Inoculum NP  
- Chimera-Breaker  
- Any dashboard, pipeline, or service

These belong to the **Applicative Layer (userland)** and are **not part of the kernel**.

## Open Source Rationale

SYF Core is open because **laws must be verifiable**.  
This repository contains only what is necessary to **reproduce and audit the law**.

> "If it has intent, it's not kernel."
